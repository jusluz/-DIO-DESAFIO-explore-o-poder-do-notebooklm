# 📘 Miniguia de Estudo

---

## 1. 📌 Fundamentos do VPA

### Harmonia
- Preço forte + volume alto

### Anomalias
- Preço forte + volume baixo → falso movimento
- Preço fraco + volume alto → absorção

---

## 2. 🏦 Ciclo dos Insiders

### Acumulação
Compra silenciosa em fundo

### Alta (Markup)
Entrada do varejo (FOMO)

### Distribuição
Venda institucional no topo

---

## 3. 🌍 Visão Relacional

Combine:
- Fundamental
- Técnico (VPA)
- Fluxo de capital

Exemplo:
```
Bonds → Ações → Commodities → Forex
```

---

# 📖 Glossário

- **VPA:** análise de preço + volume
- **Smart Money:** instituições
- **Acumulação:** compra em baixa
- **Distribuição:** venda em alta
- **Tick Volume:** proxy no Forex
- **Velas Sumo:** absorção institucional
- **Clímax:** volume extremo antes da reversão
